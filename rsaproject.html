The purpose of the program is to encrypt and decrypt messages. The user enters a message they want to encrypt along with the values of their public key to encrypt the message. Before they encrypt they can save it to a text file so they can remember the original text before they encrypt. If the person wants to decrypt the message then they’ll have to input their private key. The user inputs integers from their public key and a text message they want to encrypt. The output is an encrypted version of the message. If they want to decrypt the message back to normal they have to input integers that are from their private key and it’ll return the decrypted version or the regular input.


The function grabs the message from the input box. I take a value then numerize, then take the power of the integer, then characterize. In this case I added the dictionary code so that if we have a long message with repeating letters it’ll be easier to output since the dictionary has it stored.
